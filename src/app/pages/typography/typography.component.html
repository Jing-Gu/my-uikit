Typography

A typescale that will cover all your UI design needs

General rule of typography for this system:

  • All titles use font Montserrat
  • All body texts use font Poppins
  • To be scalable in a consistant way, each title and body has a smaller version for mobile use

Display

Display is reserved for landing pages, marketing pages where titles can be extremely bold and large.

Titles and subtitles

All titles and subtitles use font Montserrat.

To be scalable in a consistant way, each title has a smaller version for mobile use. For example, use H2 on desktop and use H2 small for the same place on mobile. The smaller H2 size is actually the normal H3 size, but keep each title for its own purpose for the sake of SEO and semantic.

A good headline size for small screens might be somewhere between 20px and 24px.
